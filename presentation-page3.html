Что такое база данных?

В общем случае база данных - это программа, позволяющая другим программам выполнять операции с данными. Типичными операциями с данными являются четыре операции, известные под акронимом CRUD:

  • Create - создание, то есть добавление некоторых данных в базу данных;
  • Read - чтение, то есть получение некоторых данных из базы;
  • Update - обновление, то есть изменение некоторых существующих данных в базе;
  • Delete - удаление некоторых данных из базы.
Таким образом, по сути базе данных необходимо следующее:
  • некоторое хранилище, обычно в виде файла или файлов на жестком диске, где она сохраняет данные в том формате, который сочтет оптимальным;
  • некоторый интерфейс для взаимодействия с ней других программ.
